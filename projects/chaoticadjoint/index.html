Adjoint and Tangent Systems for Chaotic Flows by Equation Transformation

This project is an attempt to tackle adjoint sensitivities of chaotic primal flows. The motivation is cases where the long time averaged objective function of an unsteady simulation entering limit cycle oscillation is essentially ergodic with ill-behaved sensitivity values. These problems are defined by the large scale behavior of the objective function being well behaved but with micro oscillations on the smaller scale of the design variable perturbations. This sort of behavior is shown in the figure below.

This project showed attempts to solve this problem by decomposing the unsteady governing equations into mean and fluctuating components and time averaging the equations to obtain a steady-state equation that has sensitivities that correspond well to the unsteady equations. This is similar to using the RANS adjoint to obtain sensitivities of the LES primal flow for good turbulence and filtering models.
